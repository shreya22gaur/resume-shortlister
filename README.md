**AI Resume Shortlister using Generative AI**

Developed an AI-powered Resume Shortlister that evaluates candidate resumes against a given job description using Generative AI. The system analyzes skills, experience, education, and project relevance to determine how well a candidate matches the job requirements.

The application generates an ATS (Applicant Tracking System) compatibility score and provides detailed feedback explaining why a resume is likely to be selected or rejected. It identifies matching skills, highlights missing requirements, and suggests improvements to increase the candidate's chances of passing the screening process.

**Key Features:**

* Resume and Job Description analysis
* ATS score generation
* Skill matching and gap identification
* Selection/Rejection reasoning
* Personalized resume improvement suggestions
* PDF resume processing
* Natural language feedback using Large Language Models (LLMs)

**Technologies Used:**
Python, Generative AI, LLM APIs, PDF Processing, Prompt Engineering, Google Colab, Streamlit (optional)

**Outcome:**
The tool helps job seekers understand how recruiters and ATS systems evaluate resumes, enabling them to optimize their applications and improve interview selection chances.
